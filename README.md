# Desafio_final
Este proyecto forma parte del desafío Telecom X - Parte 2, donde el objetivo principal es desarrollar un modelo de Machine Learning capaz de identificar proactivamente a los clientes con mayor probabilidad de cancelar sus servicios.


La retención de clientes es más económica que la adquisición de nuevos. En el sector de telecomunicaciones, el Churn (fuga de clientes) impacta directamente en los ingresos. Este modelo permite a la empresa anticiparse y ejecutar estrategias de fidelización personalizadas


Lenguaje: Python 3.x

Librerías: * Pandas & NumPy (Manipulación de datos)

    Scikit-Learn (Modelado y Evaluación)

    Imbalanced-learn (Balanceo con SMOTE)

    Matplotlib & Seaborn (Visualización)
